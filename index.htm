Tag "html"

Ela é serve para especificar para o navegador que será feito um documento formatado em html

Tag "head"

Dentro dela há todos os componentes que devem ser carregados antes da página ser carregada para o usúario

Tag "Body"

Ela agrega todo o conteúdo que será apresentado ao Client (usuário)

Tag "Strong"

Deixa o texto em negrito

Tag "I"

Deixa o texto em italico

Atributos de Tags

  • São configurações especificas que podem ser aplicadas nas tags.

Exemplos:

Atributo "ID"

Esse atributo server como um indentificador para a tag, isso é muito útil para quando queremos linkar um item a outro cdentro da mesma página com a tag "a".

Atributo "Style"

Ele é uma mão na roda na hora de estilizar algum elemento direto no html. Como exemplo usei a tag "mark" para marcar o texto e depois usei o atributo "style" paras trocar a cor amarela padrão pela orangered e o texto para branco (você pode conferir como eu fiz ossp dando um f12 e inspecionando o elemento "exemplo" pelas opções do desenvolvedor.

Atributo "Class"

Esse elemento (atributo) também muito é útil para estilização de tags via CSS, ele aplica as mesmas configuração de estilo em todos os elementos que estivere dentro da tag

Tag de listas ordenadas 'OL' e não ordenadas "UL"

As listas ordenadas, basicamente, são listas que seguem o padrão de numeração: 1, 2, 3...

  1. Item 1
  2. Item 2
  3. Item 3

As listas não ordenadas são o oposto das ordenadas, elas são usadas principalmente em sumários e menus:

  • Item 1
  • Item 2
  • Item 3

*Para funcionar é necessário colocar a tag "li" (lista de itens)
